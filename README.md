# Medical Record Editor (MRE) App

Ez egy egyszerű React + TypeScript alapú orvosi jegyzetelő alkalmazás.

## Futtatás fejlesztői módban

```bash
npm install
npm start
```

## Fő oldalak

- Welcome
- Register
- Login
- Notes
- Editor

## Követelmények

- Node.js
- npm
